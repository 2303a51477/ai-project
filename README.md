# 📸 QuantumPix – Photography Booking System

QuantumPix is a full-stack web application built using **Flask** and **MySQL** that allows users to book photographers, select camera equipment, and manage photography sessions for various events like weddings, birthdays, corporate shoots, and more.

---
Run this command inside your project folder:

python app.py

You should see:

Running on http://127.0.0.1:5000/

## 🚀 Features

- 🔐 User authentication (Register / Login / Logout)
- 📅 Photography booking system
- 🎯 Multiple event types (Wedding, Birthday, Fashion, Corporate, etc.)
- 📷 Camera selection with pricing
- 👨‍💼 Photographer selection
- 💰 Dynamic price calculation
- 📊 User dashboard with recent bookings
- 🧾 Booking confirmation page
- 🛠 Admin panel (view users, bookings, photographers)
- 🎨 Responsive UI templates

---

## 🏗️ Tech Stack

- **Backend:** Flask (Python)
- **Database:** MySQL
- **Authentication:** Werkzeug Security (Password Hashing)
- **Frontend:** HTML, CSS, Bootstrap (Jinja2 Templates)
- **ORM/Connector:** Flask-MySQLdb / PyMySQL
